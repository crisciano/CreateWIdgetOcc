# Create Widget Occ

Create wizard widget Oracle Commerce Cloud.

# Overview
Create structure, create files, zip widget but delete dir.

## Install Dependences

~~~javascript
npm install
~~~
## Start
~~~javascript
npm start
~~~

## Questions
